# 🐍 Python OOP Architecture Labs

This repository is a comprehensive collection of Python projects focused on Object-Oriented Programming (OOP) and software system design.

### **What I Built**
I built several mini-systems to showcase core programming principles:
* **Game Asset Manager:** A system to organize video game components (Characters, Environments, and Audio), allowing a user to search and filter assets dynamically.
* **Vehicle Allocation System:** A fleet management simulation that categorizes different transport types (Cars, Trucks, Motorcycles) and handles their specific behaviors.
* **Employee Hierarchy:** An HR simulation that manages different staff roles (Managers vs. Developers), demonstrating how different employee types calculate their bonuses and handle responsibilities.

---

## 🛠️ Technical Concepts Mastered

This portfolio heavily utilizes advanced Python architectural concepts:
* **Inheritance:** Creating base classes (like `Asset` or `Vehicle`) and extending them into specific types (like `Character` or `Motorcycle`).
* **Polymorphism & Method Overriding:** Customizing how shared methods behave differently depending on the object type (e.g., how a Developer's bonus is calculated versus a Manager's).
* **Composition & Iteration:** Building "Manager" classes that store, filter, and interact with lists of complex data objects.
